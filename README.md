## Quick Start

1. Clone this repo
   git clone https://github.com/yourname/whale-trucker-command-center

2. Setup config
   cp config.example.js config.js

3. Add your API key in config.js
   const CONFIG = {
     COINGECKO_API_KEY: "your-key-here"
   }

4. Open index.html in browser — done.
